# SIC-2026 — Aula 1

Na primeira aula da SIC 2026, tivemos uma introdução ao tema **Inteligência Artificial Generativa**. Durante a aula, discutimos questões como **o que é IA generativa**, **como ela funciona**, **quais são seus principais tipos**, além dos requisitos e aplicações dessa tecnologia. Como atividade prática, o professor propôs o seguinte desafio:

> Utilize uma ferramenta de geração de código de sua preferência para criar uma página pessoal. Registre o prompt utilizado no README e publique o código gerado em um repositório no GitHub. Em seguida, envie o link do repositório como resposta.

O objetivo da atividade foi familiarizar os alunos com ferramentas de IA generativa voltadas ao desenvolvimento de software, proporcionando uma experiência prática, independentemente de já possuírem ou não contato prévio com esse tipo de tecnologia.

Para desenvolver a atividade, escolhi utilizar o **Claude AI**, tanto pela familiaridade que já possuo com a ferramenta quanto por considerá-la uma excelente opção para geração de código. O prompt utilizado foi o seguinte:

> Crie uma página de apresentação para mim utilizando HTML, CSS e, se necessário, JavaScript. A paleta de cores deve seguir um tema escuro inspirado no GitHub, com as informações organizadas de forma criativa, utilizando elementos como cards para apresentar minhas stacks e conhecimentos. As informações da página devem ser baseadas no seguinte conteúdo:
>
> "Técnico em Eletrotécnica formado pela E.E.E.P. Professora Abigail Sampaio, atualmente cursando Técnico em Automação Industrial e Tecnólogo em Mecatrônica Industrial no Instituto Federal de Educação, Ciência e Tecnologia do Ceará (IFCE), Campus Pecém. Possui experiência na área de manutenção eletrônica, tendo atuado em uma microempresa de refrigeração, onde aprofundou seus conhecimentos em reparos e circuitos eletrônicos. Participou de projetos de pesquisa, iniciação científica e extensão, desenvolvendo habilidades em design, projeto e fabricação de placas de circuito impresso (PCI), além de programação de microcontroladores ESP e Arduino aplicados à robótica, Internet das Coisas (IoT) e sistemas inteligentes. Além da atuação na área eletroeletrônica, possui interesse e experiência em desenvolvimento web, com conhecimentos em HTML, CSS, Java, JavaScript, React, Node.js e PostgreSQL. Busca constantemente integrar tecnologia, automação e software para o desenvolvimento de soluções inovadoras."

O resultado gerado foi bastante satisfatório, apresentando uma interface moderna, organizada e com um visual profissional. A página pode ser visualizada de duas formas: Pelo arquivo `index.html`, disponível na raiz deste repositório ou pelo deploy realizado utilizando o **GitHub Pages**.
